# Simple Weather App 🌤️

A lightweight frontend weather application built with **HTML, CSS, and JavaScript**.  
It fetches live weather data using the [OpenWeatherMap API](https://openweathermap.org/api).

## Features
- Search weather by city name
- Displays temperature and conditions
- Clean, responsive UI
